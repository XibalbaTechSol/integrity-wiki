# DID (Decentralized Identifier)

This page is a redirect to the core concept: [Hardware Fingerprinting](hardware-fingerprinting.md).

A W3C-compliant **Decentralized Identifier (DID)** binds an agent's digital signature keys directly to its physical server host to guarantee non-repudiation.

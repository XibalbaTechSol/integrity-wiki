# BCC (Behavioral Commitment Chain)

This page is a redirect to the core concept: [Behavioral Commitment Chain](behavioral-commitment-chain.md).

The **Behavioral Commitment Chain (BCC)** is a cryptographic security protocol designed to force stochastic LLM agents to commit to an intended state before executing mutations.
